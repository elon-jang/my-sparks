# Mac 키보드 심볼(⇧⌃⌥⌘)의 유래와 의미

## Summary

Mac 키보드의 modifier key 심볼들은 각각 고유한 역사적 배경과 의미를 가지고 있다. 일부는 심볼과 기능의 의미가 직접 연결되어 있고, 일부는 기술적 관례나 디자인적 선택에서 유래했다.

## Key Points

- **⇧ Shift**: 타자기 시대 유래. Shift를 누르면 활자판이 물리적으로 위로 이동(shift up)해서 대문자를 찍었음
- **⌃ Control**: ASCII 제어 문자 표기법에서 유래. ^C, ^V처럼 캐럿(^)으로 Control 조합을 표기하던 관례
- **⌥ Option**: 철도 분기점 모양. 선로가 두 갈래로 나뉘듯 "대안(alternative)을 선택"한다는 의미
- **⌘ Command**: 스웨덴 캠프장의 "명소 표시" 기호에서 유래. 스티브 잡스가 사과 로고 남용을 싫어해서 Susan Kare가 채택

### 의미 연결 정도

| 심볼 | 키 | 의미 연결 |
|------|-----|-----------|
| ⇧ | Shift | 직접 연결 (위로 이동) |
| ⌥ | Option | 직접 연결 (분기/선택) |
| ⌃ | Control | 기술적 표기 관례 |
| ⌘ | Command | 역사적/디자인적 선택 (약함) |

## Questions

### Q1: Shift 키 심볼(⇧)이 위쪽 화살표 모양인 이유는?
**A**: 타자기 시절, Shift를 누르면 활자판이 물리적으로 위로 이동(shift up)해서 대문자를 찍었기 때문

### Q2: Option 키 심볼(⌥)이 철도 분기점 모양인 이유는?
**A**: 선로가 두 갈래로 나뉘듯 "대안(alternative)을 선택"한다는 의미를 표현

### Q3: Command 키 심볼(⌘)의 유래는?
**A**: 스웨덴 캠프장의 "명소 표시" 기호. 스티브 잡스가 사과 로고 남용을 싫어해서 Susan Kare가 채택
