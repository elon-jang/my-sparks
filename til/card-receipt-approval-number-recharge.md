# 카드 영수증 분실 시 앱으로 승인번호 확인하여 재결제하기

## Summary

카드 결제 후 영수증을 분실했어도, 카드사 앱에서 승인번호만 확인하면 매장에서 결제 취소 및 재결제가 가능합니다. POS기에서 결제 취소 시 필요한 것은 종이 영수증이 아닌 '승인번호'입니다.

## Key Points

- 결제 취소에 필요한 것은 승인번호(8자리 숫자)뿐
- KB국민카드 앱: [카드이용내역 조회] → [이용내역(매출전표)] → 결제 건 선택 → 승인번호 확인
- 매출전표 화면을 캡처해서 매장에 가면 편리
- 매장 방문 시 기존 카드(취소용) + 새 카드(결제용) 실물 필수

## Questions

### Q1: 영수증 없이 카드 결제를 취소하려면 무엇이 필요한가?

**A**: 승인번호(8자리 숫자)만 있으면 됩니다. 종이 영수증은 승인번호를 보여주는 도구일 뿐이므로, 카드 앱에서 승인번호를 확인하면 취소 가능합니다.

### Q2: KB국민카드 앱에서 승인번호를 확인하는 경로는?

**A**: [카드이용내역 조회] → [이용내역(매출전표)] → 해당 결제 건 터치 → 상세 화면 하단의 8자리 승인번호 확인

### Q3: 카드 재결제를 위해 매장 방문 시 필수로 챙겨야 할 것은?

**A**: 기존 카드(취소용)와 새 카드(결제용) 실물 두 장 모두 챙겨야 합니다. 삼성페이 등으로 취소가 안 될 경우 실물 카드가 필요할 수 있습니다.
