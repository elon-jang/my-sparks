# 구슬샘 영어 웹앱 출시! 총정리 영상으로 영어 표현 마스터하기

구슬샘의 인기 유튜브 콘텐츠 "총정리" 시리즈를 더 효과적으로 학습할 수 있는 웹앱이 출시되었습니다. 영상 속 50개 표현을 구간별로 반복 학습하고, 진도를 체크하며, 나만의 속도로 완주해보세요.

---

## 왜 이 앱을 만들었나요?

구슬샘의 총정리 영상은 50개 표현이 빠르게 지나갑니다. "이 표현 뭐였지?" 하고 돌려보려면 정확한 타임스탬프를 찾아야 하고, 어디까지 봤는지 기억하기도 어렵죠.

이 앱은 그 문제를 해결합니다:
- 각 표현별로 **정확한 영상 구간만** 재생
- **학습 완료 체크**로 진도 관리
- **반복 재생 모드**로 귀에 박힐 때까지

---

## 메인 화면 살펴보기

### 1. 영상 플레이어 영역

화면 상단에서 현재 표현 **"hung up on"**이 크게 표시됩니다. 가운데 **재생 버튼**을 누르면 이 표현이 나오는 영상 구간만 딱 재생됩니다.

**"1회 재생" 버튼**을 누르면 반복 재생 모드로 전환할 수 있어요. 듣고 또 듣고, 귀에 익을 때까지!

### 2. 학습 진도 바

**"학습 진도 0%"** — 완료 체크한 표현의 비율이 표시됩니다. 50개 중 25개를 완료하면 50%가 되겠죠.

### 3. 표현 카드

각 표현은 3가지 정보로 구성됩니다:

| 섹션 | 설명 |
|------|------|
| **SITUATION** (보라색) | 이 표현을 언제 쓰는지 상황 설명 |
| **EXAMPLE** (핑크색) | 실제 예문과 한국어 번역 |
| **MORE** | 추가 예문 (있는 경우) |

오른쪽 **체크 버튼(✓)**을 누르면 이 표현을 "학습 완료"로 표시할 수 있습니다.

## 전체 표현 목록

헤더 오른쪽 **햄버거 메뉴(≡)**를 누르면 전체 50개 표현 목록이 나타납니다.

### 한눈에 보는 학습 현황

- **50** 전체 | **0** 완료 | **50** 남은 표현

### 검색 기능

"표현이나 의미 검색..." 필드에 키워드를 입력하면 원하는 표현을 바로 찾을 수 있습니다. "get"을 검색하면 "get over", "can't get over", "get out of your hair" 등이 필터링됩니다.

### 표현 선택

목록에서 원하는 표현을 탭하면 해당 카드로 바로 이동합니다. 현재 보고 있는 표현은 **핑크색 테두리**로 하이라이트됩니다.

---

## 설정 메뉴

헤더의 **톱니바퀴 아이콘(⚙️)**을 누르면 설정 화면이 열립니다.

### 회차 선택

현재는 **45탄**이 제공됩니다. 앞으로 더 많은 회차가 추가될 예정이에요!

### 재생 설정

| 설정 | 설명 |
|------|------|
| **자동 재생** | 표현 이동 시 자동으로 영상 재생 (ON 추천) |
| **재생 속도** | 0.75x, 1x, 1.25x 중 선택 |

처음에는 1x로 듣다가, 익숙해지면 1.25x로 도전해보세요!

### 학습 데이터 초기화

모든 완료 체크를 지우고 처음부터 다시 시작하고 싶을 때 사용합니다.

## 추천 학습 방법

### 1단계: 전체 훑어보기
처음에는 자동 재생을 켜고 스와이프하면서 50개 표현을 쭉 훑어보세요. "아, 이런 표현들이 있구나" 감 잡기용입니다.

### 2단계: 구간 반복
모르는 표현이 나오면 **루프 모드**를 켜고 여러 번 들어보세요. 발음과 억양이 귀에 익을 때까지!

### 3단계: 완료 체크
확실히 알겠다 싶은 표현은 **체크 버튼**을 눌러 완료 표시. 진도율이 올라가는 걸 보면 뿌듯합니다.

### 4단계: 복습
며칠 후 다시 들어와서 완료 표시한 것들만 빠르게 훑어보세요. 기억나면 OK, 아니면 체크 해제하고 다시 학습!
